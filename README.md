# reactale-v2
Reactale FE (ReactJS) and BE (Node.js) in one project

# Backup Strategy 2021

- Connect with WinSCP FTP client to the server
- Copy the user-upload/banner folder
- On server, use `mongodump` to create the dump folder, and then copy that folder over ftp too
- Everything is saved in GoogleDrive of reactale@gmail.com
